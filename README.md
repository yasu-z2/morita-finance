# 📈 Morita-Finance: 株価選別システム スタンドアロン版（v1.11）

東証プライム市場から「底値圏からの初動銘柄」を自動検出し、具体的な投資判断を支援するツールです。

---

## 🚀 本プロジェクトの目的
個人の力では追いきれない約1,600銘柄の東証プライム銘柄を毎日フルスキャン。
「誰にも注目されていない底値」から「大口の資金が流入した瞬間」を数学的に特定し、投資効率を最大化します。

## 🛠 主な機能
- **2段階スクリーニング**: 
  - `第一段階(実戦)`: 勢いのあるリバウンド初動を抽出。
  - `第二段階(厳格)`: 25日間の完全底這い＋出来高激増の「究極の初動」を特定。
- **自動指値計算**: 現在値や安値から、統計的に有利な「第1・第2指値」を自動算出。
- **マルチレポート出力**: 
  - データ分析用の `CSV` ファイル。
  - 人間が読みやすいレイアウトの `TXT` レポート。

## 📋 クイックスタート

### 準備
1. [Python](https://www.python.org/) をインストール（インストール時に `Add Python to PATH` をチェック）。
2. 本リポジトリをダウンロードし、フォルダ内の `setup_minimal.bat` を実行。

### 運用
1. `data_jpx.csv` (JPX公式サイトから取得) をフォルダに配置。
2. `run_scan.bat` を実行して、その日のレポートを確認。
