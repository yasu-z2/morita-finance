# 株価選別・AI分析システム (v3.2.5)

東証プライム銘柄から「底値圏からの初動」を捉え、最新の **Gemini 2.5 Flash** が投資戦略を立案する自動配信システムです。

## 🚀 主な機能

1. **Gemini 2.5 Flash 搭載**:
   Google最新モデルによる「市場俯瞰」と「具体的な立ち回り戦略」の二段構え分析。
2. **ロジックの可視化**:
   配信メール内に各段階の抽出条件を明記し、銘柄選定の根拠を明確化。
3. **ソースリンク自動付与**:
   銘柄ごとに **Yahoo!ファイナンス** および **株探** へのダイレクトリンクを付与。
4. **差分キャッシュ高速化**:
   GitHub Actionsのキャッシュ機能を活用し、全プライム銘柄のスキャンを約4分で完了。

---

## ⚙️ 判定ロジック詳細

レポート内では以下の条件に基づき銘柄を分類しています。

### ▼▼ 【第一段階：実戦モード】 注目候補 ▼▼
- **底値圏**: 過去25日安値から **+15%以内**
- **初動**: 当日終値が安値から **+10%以上** 上昇
- **出来高**: 当日2.0倍、前日1.5倍以上の急増

### ▼▼ 【第二段階：厳格モード】 特選初動候補 ▼▼
※上記の中からさらに厳選
- **底値圏**: 安値から **+10%以内**
- **出来高**: **2日連続で2.0倍以上** の急増を記録した最有力候補

## 🛠 セットアップ (GitHub Secrets)

- `GEMINI_API_KEY`: Google AI Studio APIキー
- `MAIL_ADDRESS`: 送信元Gmail
- `MAIL_PASSWORD`: Gmailアプリパスワード
- `TO_ADDRESS`: 送信先（カンマ区切りで複数指定可能）
