# 株価選別・AI分析システム (v3.2.7)

東証プライム銘柄の中から、「底値圏」かつ「出来高急増」を伴う「初動銘柄」を自動抽出し、Gemini 2.5 Flashによる分析をメール配信するシステムです。

## 抽出条件

### 【第一段階：注目候補】
1. **底値圏**: 過去25日間の最安値から、現在値が **+15%以内**。
2. **初動**: 当日の安値から終値が **+10%以上** 上昇（強い陽線）。
3. **出来高**: 当日出来高が過去25日平均の **2.0倍以上**、かつ前日も **1.5倍以上**。

### 【第二段階：特選初動候補】
第一段階を通過した銘柄のうち、さらに以下を満たすもの。
- **底値圏**: 過去25日安値から **+10%以内**。
- **出来高**: 2日連続で **2.0倍以上** の急増。

## アップデート内容 (v3.2.7)

- **統計メール送信**: 該当銘柄が0件の場合でも、判定の各段階で何銘柄残ったかの統計情報を添えてメールを送信します。
- **AIモデル更新**: 分析エンジンに最新の `gemini-2.5-flash` を採用。
- **初動ロジックの修正**: 「当日安値から終値への上昇率」を判定基準に設定し、1日の足の強さを重視。
- **UI改善**: メールの表題と説明文をより直感的な表現に変更。

## 設定方法
GitHub Secretsに以下の環境変数を設定してください。
- `GEMINI_API_KEY`: Google AI Studioから発行されたキー
- `MAIL_ADDRESS`: 送信元Gmailアドレス
- `MAIL_PASSWORD`: Gmailアプリパスワード
- `TO_ADDRESS`: 通知先メールアドレス（カンマ区切りで複数指定可）
