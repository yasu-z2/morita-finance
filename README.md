# 株価選別・AI分析システム (v3.2.2)

このプロジェクトは、東証プライム銘柄から「底値圏からの初動」を捉えるスクリーニング・システムです。ローカル検証用の「スタンドアロン版」と、自動運用・AI分析を行う「AIメール配信版」の2つのモードを備えています。

## 📊 システムの全体像
本システムは、共通の判定アルゴリズム（v1.11）を使用しており、検証と本番で結果が完全に一致するように設計されています。



---

## 1. スタンドアロン版 (検証・ローカル実行)
PC上で直接実行し、詳細な検証データを確認するためのモードです。

### 特徴
- **詳細出力**: 全スキャン結果を `Report_yyyymmdd.csv` に保存。
- **即時確認**: 実行後すぐに `Report_yyyymmdd.txt` で判定結果を閲覧可能。
- **柔軟性**: パラメータ（レンジ幅、出来高倍率など）の調整・テストに最適。

### 実行方法
```bash
python scanner_ai.py  # (または検証用スクリプト名)
